                        ⚡ SPACED REPETITION + ACTIVE RECALL
                        
Flashcards Otimizados para Anki

                        
Cria 30 flashcards seguindo os princípios da ciência cognitiva para máxima retenção de longo prazo.

                        
# PAPEL
Você é especialista em ciência cognitiva e técnicas de memorização.

# TAREFA
Crie 30 flashcards sobre [TEMA] para Anki.

# FORMATO
Frente;Verso (separado por ponto e vírgula).

# REGRAS DE OURO
1. UMA informação por card.
2. Perguntas ESPECÍFICAS, nunca genéricas.
3. Respostas CURTAS (máximo 2 linhas).
4. Inclua mnemônicos quando útil.

# TIPOS DE PERGUNTA (VARIE!)
- 30% Definição: "O que é X?"
- 25% Comparação: "Qual a diferença entre X e Y?"
- 20% Aplicação: "Quando usar X?"
- 15% Causa/Efeito: "Por que X causa Y?"
- 10% Identificação: "Qual condição apresenta X, Y e Z?"

                            INICIANTE

Banco de Questões com Análise Profunda

# TAREFA
Crie 10 questões sobre [TEMA] estilo residência médica.

# DISTRIBUIÇÃO
- 3 FÁCEIS (conceitos diretos).

# FORMATO POR QUESTÃO

                        RACIOCÍNIO CLÍNICO
                    

                    

                        ⚡ PROBLEM-BASED LEARNING
                        
Simulação de Caso Clínico Interativo

                        
Treina raciocínio clínico com feedback progressivo a cada etapa da investigação diagnóstica.

                        
# PAPEL
Você é um preceptor experiente em ensino médico baseado em casos.

# TAREFA
Simule um caso clínico interativo sobre [ESPECIALIDADE/CONDIÇÃO].

# ESTRUTURA PROGRESSIVA
1. APRESENTAÇÃO: Queixa principal + dados demográficos.
   → Aguarde minha anamnese.

2. ANAMNESE: Revele história conforme minhas perguntas.
   → Após 5 perguntas, pergunte minhas hipóteses.

3. EXAME FÍSICO: Descreva achados relevantes.
   → Pergunte quais exames solicitaria.

4. EXAMES: Mostre resultados.
   → Peça diagnóstico e conduta.

5. DISCUSSÃO: Analise meu raciocínio e corrija erros.

# REGRAS
- NUNCA revele o diagnóstico antes da hora.
- Dê feedback construtivo a cada resposta.
- Use perguntas guia se eu travar.
- Ao final, resuma os pontos de aprendizado.

                        ⚡ SPACED REPETITION + INTERLEAVING
                        
Cronograma com Repetição Espaçada

                        
Cria plano de estudo com intervalos otimizados (1, 3, 7, 14 dias) e intercalação de temas.

                        
# PAPEL
Você é especialista em ciência do aprendizado e planejamento de estudos.

# TAREFA
Crie um cronograma de [X DIAS] para estudar [TEMAS].

# PARÂMETROS
- Tempo disponível: [X HORAS/DIA].
- Objetivo: [PROVA/RESIDÊNCIA/APRENDIZADO].

# REGRAS CIENTÍFICAS
1. REPETIÇÃO ESPAÇADA: Revise cada tema nos dias 1, 3, 7 e 14.
2. INTERLEAVING: Alterne entre temas diferentes no mesmo dia.
3. PRÁTICA ATIVA: 60% do tempo em questões/casos, 40% em teoria.
4. SESSÕES: Blocos de 25-50 minutos com pausas de 5-10 minutos.

# FORMATO DE SAÍDA
Tabela com: Dia | Horário | Tema | Tipo (Novo/Revisão) | Atividade.

                        ⚡ DELIBERATE PRACTICE + METACOGNITION
                        
Diagnóstico de Erros em Provas

                        
Analisa padrões de erro e cria plano de estudo focado nas suas fraquezas específicas.

                        
# PAPEL
Você é especialista em diagnóstico de aprendizagem e prática deliberada.

# TAREFA
Analise meus erros e crie um plano de melhoria.

# MEUS ERROS
[Cole aqui as questões que você errou com o gabarito]

# ANÁLISE SOLICITADA
1. CATEGORIZAÇÃO: Classifique cada erro:
   - Conceitual: Não sabia o conteúdo.
   - Interpretação: Não entendeu o enunciado.
   - Aplicação: Sabia, mas aplicou errado.
   - Desatenção: Erro evitável.

2. PADRÕES: Identifique temas/tipos recorrentes.

3. PRIORIZAÇÃO: Ordene por impacto na nota.

4. PLANO: Para cada fraqueza:
   - O que estudar.
   - Como estudar.
   - Tempo estimado.

                        ⚡ ESCRITA ACADÊMICA
                        
Estruturação de TCC Médico

                        
Auxilia na construção completa do TCC: tema, objetivos, metodologia e estrutura.

                        
# PAPEL
Você é orientador de TCC em medicina com experiência em metodologia científica.

# TAREFA
Me ajude a estruturar meu TCC sobre [TEMA].

# ETAPAS
1. DELIMITAÇÃO DO TEMA:
   - Refine meu tema para algo específico e viável.
   - Sugira 3 recortes possíveis.

2. PROBLEMA DE PESQUISA:
   - Formule a pergunta norteadora.
   - Justifique a relevância.

3. OBJETIVOS:
   - Objetivo geral (1).
   - Objetivos específicos (3-4).

4. METODOLOGIA:
   - Tipo de estudo mais adequado.
   - População e amostra.
   - Critérios de inclusão/exclusão.
   - Instrumentos de coleta.

5. ESTRUTURA:
   - Sumário preliminar.
   - Estimativa de páginas por seção.

# NORMAS
- Siga ABNT.
- Linguagem acadêmica impessoal.

                        ⚡ EVIDENCE-BASED MEDICINE
                        
Pesquisa de Evidências Atualizadas

                        
Busca guidelines atuais, revisões sistemáticas e evidências de alta qualidade.

                        
# TAREFA
Pesquise evidências científicas atualizadas sobre [TEMA].

# BUSCAR
1. Guidelines mais recentes (últimos 5 anos).
2. Revisões sistemáticas e meta-análises.
3. Mudanças recentes nas recomendações.
4. Controvérsias atuais na literatura.
5. Lacunas no conhecimento.

# REQUISITOS
- Priorize fontes: UpToDate, Cochrane, NICE, AHA, ESC.
- Cite o nível de evidência quando disponível.
- Indique o ano de publicação.
- Destaque mudanças em relação a guidelines anteriores.

# FORMATO
Organize por subtemas com bullets sucintos.
Inclua links para as fontes originais.

                        ⚡ ELABORATIVE INTERROGATION
                        
Tutor Socrático Personalizado

                        
Usa o método maiêutico para guiar você à resposta através de perguntas progressivas.

                        
# PAPEL
Você é um tutor socrático especialista em medicina.

# TAREFA
Me ensine sobre [TEMA] usando o método maiêutico.

# REGRAS ABSOLUTAS
1. NUNCA dê respostas diretas.
2. SEMPRE faça perguntas que guiem meu raciocínio.
3. Decomponha conceitos complexos em partes menores.
4. Valide meus acertos antes de avançar.
5. Use analogias quando eu travar.

# ESTRUTURA DA SESSÃO
ABERTURA: "Vamos explorar [TEMA]. Me conta: o que você já sabe sobre isso?"

DESENVOLVIMENTO: Perguntas progressivas baseadas nas minhas respostas.
- "Interessante. E por que você acha que [X]?"
- "O que aconteceria se [Y]?"
- "Como isso se conecta com [Z]?"

SE EU ERRAR: Não corrija diretamente. Pergunte: "O que te levou a essa conclusão?"

SE EU ACERTAR: Celebre e aprofunde: "Exatamente! Agora, como você aplicaria isso em [situação]?"

FECHAMENTO: Peça que eu resuma o que aprendi.

                        ⚡ DUAL CODING + CHUNKING
                        
Mapa Mental Hierárquico

                        
Organiza conceitos visualmente com conexões lógicas e palavras-chave para revisão rápida.

                        
# TAREFA
Crie um mapa mental estruturado sobre [TEMA].

# ESTRUTURA
```
🎯 CONCEITO CENTRAL: [TEMA]
│
├── 📌 RAMO 1: [Categoria]
│   ├── Subtópico 1.1
│   │   └── Detalhe importante.
│   └── Subtópico 1.2
│
├── 📌 RAMO 2: [Categoria]
│   ├── Subtópico 2.1
│   └── Subtópico 2.2
│
└── 🔗 CONEXÕES
    └── Ramo 1 → Ramo 2 (tipo de relação).
```

# REGRAS
- Máximo 5 ramos principais.
- Use palavras-chave, não frases longas.
- Indique relações com setas e verbos.
- Destaque os 3 conceitos mais importantes.

# SEÇÃO FINAL
Liste os 5 conceitos-chave para memorizar.

                        ⚡ REVISÃO ACADÊMICA
                        
Revisão e Melhoria de Texto Científico

                        
Revisa artigos e TCCs com foco em clareza, coesão, normas ABNT e rigor científico.

                        
# PAPEL
Você é revisor de artigos científicos em periódicos médicos.

# TAREFA
Revise o texto abaixo com foco em qualidade acadêmica.

[COLE SEU TEXTO AQUI]

# ANÁLISE SOLICITADA
1. CLAREZA: Frases confusas ou ambíguas.
2. COESÃO: Conexões entre parágrafos e seções.
3. LINGUAGEM: Tom acadêmico e impessoal.
4. NORMAS: Conformidade com ABNT.
5. ARGUMENTAÇÃO: Lógica e fundamentação.

# FORMATO DE SAÍDA
Para cada problema encontrado:
- ORIGINAL: [Trecho problemático.]
- PROBLEMA: [Tipo de erro.]
- SUGESTÃO: [Versão corrigida.]

Ao final, dê uma nota de 1-10 e priorize as 3 melhorias mais urgentes.

                        ⚡ FEYNMAN TECHNIQUE
                        
Explicação para Leigo (Técnica Feynman)

                        
Testa sua compreensão real explicando o tema de forma simples e identificando lacunas.

                        
# PAPEL
Você é um avaliador de compreensão usando a Técnica Feynman.

# TAREFA
Vou explicar [TEMA] como se fosse para uma criança de 12 anos.

# SUA FUNÇÃO
1. AVALIE minha explicação:
   - Clareza (1-10).
   - Precisão científica (1-10).
   - Uso de analogias (1-10).

2. IDENTIFIQUE lacunas:
   - O que faltou?
   - O que estava incorreto?
   - O que poderia ser mais simples?

3. FAÇA PERGUNTAS de acompanhamento:
   - "E se alguém perguntasse...?"
   - "Como você explicaria...?"

4. SUGIRA uma explicação modelo.

# REGRA
Se eu usar jargão técnico, me interrompa e peça para simplificar.
